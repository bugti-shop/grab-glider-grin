---
name: Slash shortcuts auto-run
description: Notes editor slash commands should run without requiring extra Space or Enter when the command is complete
type: preference
---
Slash shortcuts in the RichTextEditor should execute automatically as soon as safe/complete, especially mobile commands like `/today`, `/now`, headings, lists, and checklist commands. Do not regress to requiring Space + Enter for no-argument slash commands. Commands with free-text arguments may still use Space or Enter to signal completion.
